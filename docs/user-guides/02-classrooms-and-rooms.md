# Guide : Gestion des Classes et des Salles

Apprenez à organiser vos espaces physiques et vos groupes d'élèves.

## 1. Gestion des Salles (Rooms)
Rendez-vous dans l'onglet **"Salles & Disponibilité"** de la [Gestion des Classes](http://localhost:3000/admin/classrooms).

### Créer une salle
1. Cliquez sur **"Nouvelle salle"**.
2. Donnez-lui un nom (ex: Salle B201) et une capacité.
3. Choisissez le type (Classe, Laboratoire, etc.).

### Vérifier la disponibilité
Chaque carte de salle affiche une **Timeline** :
- **Vert** : La salle est libre.
- **Rouge** : Un cours est actuellement programmé dans cette salle.
- Le pourcentage indique le taux d'occupation quotidien.

## 2. Gestion des Classes (Classrooms)
Allez dans l'onglet **"Classes"**.

### Créer une classe
1. Cliquez sur **"Nouvelle classe"**.
2. Saisissez le nom (ex: 6ème A).
3. Liez-la à un **Niveau** (obligatoire pour le programme scolaire).
4. Définissez la capacité maximale.

### Inspecter une classe
Cliquez simplement sur la carte d'une classe pour ouvrir sa **Fiche Détail** :
- Voir la liste des élèves inscrits.
- Consulter l'emploi du temps spécifique à cette classe.
- Vérifier le taux de remplissage via la barre de progression.
