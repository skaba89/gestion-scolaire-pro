# Guide : Gestion de l'Emploi du Temps

Ce guide vous aide à organiser les cours hebdomadaires.

## 1. Accéder au planning
Allez sur la page [Emploi du Temps](http://localhost:3000/admin/schedule).

## 2. Filtrer la vue
Utilisez les filtres en haut de page pour trouver rapidement ce que vous cherchez :
- **Département** : Pour isoler une filière.
- **Niveau** : Pour voir toutes les classes d'un même niveau (ex: Terminales).
- **Classe** : Sélectionnez la classe spécifique pour afficher sa grille.

## 3. Ajout Manuel d'un créneau
1. Cliquez sur **"Nouveau créneau"**.
2. Choisissez la **Matière**.
3. Sélectionnez le **Jour** et l'**Heure** (début/fin).
4. Choisissez une **Salle** disponible.
5. Cliquez sur **"Ajouter au planning"**.

## 4. Génération Automatique (IA Assistant)
Pour gagner du temps, utilisez l'assistant :
1. Cliquez sur **"Auto-générer"**.
2. Sélectionnez la classe cible.
3. Choisissez les jours travaillés.
4. L'assistant distribuera les matières en fonction de leur **Coefficient** (les matières à gros coefficient auront plus d'heures).
5. Vous pouvez ensuite ajuster manuellement le résultat.

## 5. Suppression
Pour retirer un cours, survolez simplement le bloc dans la grille et cliquez sur l'icône **Poubelle**.
