# Guide : Gestion des Années Scolaires

Ce guide vous explique comment configurer et gérer les périodes académiques de votre établissement.

## 1. Accéder à la gestion
Rendez-vous sur [Années Scolaires](http://localhost:3000/admin/academic-years).

## 2. Créer une nouvelle année
1. Cliquez sur le bouton **"Nouvelle année"** en haut à droite.
2. Saisissez le **Nom** (ex: 2024-2025).
3. Sélectionnez la **Date de début** et la **Date de fin**.
4. Cochez **"Année en cours"** s'il s'agit de l'année active pour tout le système.
5. Cliquez sur **"Créer"**.

## 3. Définir l'année active
Seule une année peut être définie comme "En cours". Lorsqu'une année est marquée comme telle :
- Les nouvelles inscriptions s'y rattachent par défaut.
- L'emploi du temps affiché correspond à cette période.
- Les rapports de performance extraient les données de cette année.

## 4. Gérer les Trimestres/Semestres
Une fois l'année créée, vous pouvez définir ses subdivisions en cliquant sur **"Gérer les trimestres"**.
- Ajoutez le Trimestre 1, 2 et 3.
- Définissez les dates spécifiques à chaque trimestre.
