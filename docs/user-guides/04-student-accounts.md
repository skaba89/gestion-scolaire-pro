# Guide : Création de Comptes Élèves

Ce guide vous explique comment donner un accès portail à vos élèves.

## 1. Pourquoi les élèves ne sont pas dans la liste ?
Par défaut, un élève créé dans la "Gestion des Élèves" est un **dossier administratif**. Il n'a pas de compte utilisateur (identifiant/mot de passe) tant que vous ne l'avez pas créé.

## 2. Créer un compte depuis la Gestion des Élèves (Recommandé)
C'est la méthode la plus simple pour lier automatiquement le compte au dossier de l'élève.
1. Allez dans [Gestion des Élèves](http://localhost:3000/admin/students).
2. Repérez l'élève dans la liste.
3. Cliquez sur les **trois petits points (...)** à droite de la ligne.
4. Cliquez sur **"Créer compte"**.
   - *Note : L'élève doit avoir une adresse email renseignée dans son dossier.*
5. Un email sera envoyé à l'élève avec ses identifiants.

## 3. Créer un compte depuis la Gestion des Utilisateurs
1. Allez dans [Gestion des Utilisateurs](http://localhost:3000/admin/users).
2. Cliquez sur **"Ajouter un utilisateur"**.
3. Saisissez les informations de l'élève.
4. Dans le champ **Rôle**, sélectionnez **"Élève"**.
5. Cliquez sur **"Créer le compte"**.

## 4. Vérification
Une fois le compte créé, l'élève apparaîtra avec le badge jaune **"Élève"** dans la liste globale des utilisateurs. Il pourra alors se connecter pour voir ses notes et son emploi du temps.
